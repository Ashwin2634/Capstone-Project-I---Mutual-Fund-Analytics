# dashboard/

Contains the **interactive dashboard** for the Bluestock Mutual Fund Analytics platform.

## Files

- `bluestock_mf_dashboard.pbix` → Power BI Desktop file (main dashboard)
- `dashboard.pdf` → Static/exported version of the dashboard
- `dashboard_png/` → Screenshots and individual chart images

## Key Features

- Fund performance overview
- Portfolio optimization visuals
- Risk-return analysis
- Investor demographics
- Live NAV trends

**To view:** Open `bluestock_mf_dashboard.pbix` in Power BI Desktop.
